Character Selector v.0.0.1

Chars Annotations

Draw a rectangle around each letter. Press "C" to begin. Make sure to select each visible letter in the order in which they should be read based off of the source material. When finished, Procede to the annotation box. Select each text input, a keyboard will appear on screen to help your input. Input for all characters, if u can't find your character you can select it from the unicode selector. You can use TAB on your to easily transition to the next text box. Then once all are annotated press submit.
